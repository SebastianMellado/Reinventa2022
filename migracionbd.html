Migración Bases de Datos


El Servicio de Migración de Base de Datos de Reinventa consiste en tomar los servicios asociados a Service Desk que actualmente están en SQL Server y llevarlos a Mysql.


Alcance del Servicio

  • Análisis

    • o Revisión de la base de datos actual (Verificación de versión, motor, cotejamiento).

    o Revisión de los datos almacenados en la base de datos.

    o Revisión de conexiones con otros sistemas (EJ: herramienta Power BI).

  • Ambiente de Pruebas

    • o Back up y uploads

    o Montaje, instalación

    o Evaluaciones y correcciones

  • Proceso de Migración

    • o Conversión, correcciones

    o Evaluaciones de prueba
