Migración Service Desk


El Servicio de Migración de InvGate Service Desk consiste en llevar los servicios desde una instancia on Premise a Cloud, trabajamos con los servicios de Amazon, que considera alojamiento del servicio en servidores compartidos sin costo para nuestros clientes.


Alcance del Servicio

  • Habilitación de ambiente Cloud dentro de la nube Amazon de Invgate.
  • Habilitación de Base de datos dentro de la nube de Amazon de Invgate.
  • Creación de reglas de seguridad y conexiones para el ambiente.
  • Habilitación de licencia.
  • Extracción de la base de datos de Service Desk, enviando hacia el ambiente Cloud de Invgate.
  • Extracción de carpeta Uploads, enviando hacia el ambiente Cloud de Invgate.
  • Extracción de licencia local, enviando hacia el ambiente Cloud de Invgate.
  • Habilitación de licencias Cloud.
  • Montaje de Base de datos y Carpeta Uploads.
  • Conexión Service Desk Cloud con office 365.
  • Conexión Service Desk con Active Directory por medio de conexión VPN o IP punto a punto.

    • Beneficios

  • Facilita el trabajo de TI automatizando tareas
  • Uptime 99.99%
  • Respaldo diario, semanal y mensual
  • Alta disponibilidad
  • Reducción de costos de infraestructura y mantenimiento
  • Seguridad en infraestructura de Amazon
